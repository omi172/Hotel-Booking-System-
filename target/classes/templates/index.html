Hotel Booking System

A Spring Boot web application that allows users to search hotels, view available rooms, and make bookings — similar to Booking.com & OYO.

Search Hotels Get Started

System Modules

User Module

Secure registration & login with JWT authentication, password hashing, and role-based access control.

Hotel Module

Admins can create, update, view, and delete hotel records. Users can browse hotel listings.

Search Module

Search hotels by location and date range. Automatically filters out already booked rooms.

Booking Module

Book rooms with date conflict detection, view booking history, and cancel reservations.

Payment Module

Process payments with booking ID, amount tracking, and status management (SUCCESS/FAILED).
